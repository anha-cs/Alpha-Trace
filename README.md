# Alpha-Trace: A playful hand‑tracking tracing game for kids learning English letters and numbers.


## Why I Built This

My mom is a nanny, and she takes care of a little girl who’s almost five. She’s curious, energetic, and loves anything that feels like magic. I wanted to give her a fun way to learn English characters — something more engaging than worksheets or flashcards. 

I learned that kids remember letters better when they use big movements with the shoulder, arm, and whole‑hand working together, not just tiny pencil strokes. So this app lets her trace in the air with her whole arm, making learning faster, more natural, and a lot more like play.

The app uses the webcam and MediaPipe Hands to track the child’s fingertip in real time. As they trace each letter or number, glowing checkpoints light up, progress fills, phonics are spoken aloud, and a colorful confetti celebration appears when they finish.


## Features
- **Hand‑tracking tracing** using the webcam  
- **A–Z letters and 0–9 numbers**  
- **Phonics audio** (“A is for Apple”)  
- **Progress bar** that fills as the child traces  
- **Colorful confetti celebration** when completed  
- **Kid‑friendly UI** with bright colors and simple interactions  
- **Auto‑advance mode** for “Play All” learning sessions  


## Tech Stack
- JavaScript  
- HTML5 Canvas  
- MediaPipe Hands  
- Web Speech API  


## Running the Project
Just open `index.html` in a browser that supports webcam access.
Notes: I put everything inside 1 index.html file so it is easier to open with any browser and on any device without installing anything or setting up a development environment. Just open the file in a browser and it works.
